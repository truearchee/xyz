## What it must prove (not merely a 200)
The smoke must prove the **actual brief path returns a parseable `BriefSummary` `{"text": ...}`** —
the gateway → real provider → tolerant-extract/strict-shape validator → stored `text`. A bare 200 is
not sufficient.

## Procedure (synthetic data only)

production data is sent to IFM in 4.5b** (real transcripts wait for the controlled 4.5d path). One
shape:

```
1. Render brief_summary/v1 against the synthetic transcript via the PromptRegistry.
2. Call LLMGateway.complete(prompt_key=brief_summary/v1, output_schema=BriefSummary, priority=background,

   and run the OutputValidator on the result).
3. Assert: result["parsed"] is a BriefSummary with non-empty text; AIRequestLog row status=succeeded,
   backend_used=cerebras, backend_route_source=requested, provider_attempt_count=1.
```
