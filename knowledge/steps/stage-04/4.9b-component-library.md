## Key decisions
- **Modal → React Aria (stable)**; **Toast → owned** (RAC Toast is `UNSTABLE_*` in 1.18 — a frozen
  contract must not ride it; the toast behavior is a simple polite live region). Recorded as the ADR-046
  amendment.
- **Contract stability (ADR-047):** breaking (rename/remove) frozen post-4.9, additive allowed via the
  design-system.md changelog. As-built contracts listed in the ADR; recorded from shipped code at 4.9e.
- **Empty-State sequencing (4.9a gap) CLOSED:** error/404/403/Access-denied now consume `EmptyState`.
