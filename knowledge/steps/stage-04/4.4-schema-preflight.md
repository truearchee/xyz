Partial unique index:

```sql
CREATE UNIQUE INDEX ingestion_jobs_one_active_embed_per_transcript
ON ingestion_jobs (transcript_id, job_type)
WHERE job_type = 'embed'
  AND status IN ('queued', 'running');
```

Downgrade behavior:

- drops the embed-only partial unique index
- drops the provenance constraint
- drops only the newly added provenance columns
- does not drop the vector extension because `0006` still owns vector-backed `transcript_chunks.embedding`
