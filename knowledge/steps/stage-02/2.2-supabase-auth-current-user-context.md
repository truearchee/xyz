## Risks introduced
- `PyJWKClient` will need network access to fetch a signing key when a key is not already cached. Tests mock this path and do not call Supabase.
- The running local `.env` may still contain legacy JWT variables from earlier setup; `.env.example` now documents the required JWKS variables.
- Backend tests still surface the existing `httpx` ASGI shortcut deprecation warning.
