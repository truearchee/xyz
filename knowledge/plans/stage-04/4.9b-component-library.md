Delivers the §4.2 public components + helpers + §4.3 primitives, token-only, each with a co-located
static-a11y smoke (minimal Vitest runner here; full gates 4.9d), plus the error/404/403 → Empty State

or touch backend / `upload.ts`. Four developer holds: F-4.9-3 filed (done), React-Aria token-only
(grep-proven), a11y smoke per component, gate selectors preserved through the Empty State refactor.

## Approach
- **Contracts first, values from tokens.** Each component's variants/states/a11y match the §4.2 table
  exactly; visuals come only from ADR-045 semantic utilities. `variants.ts` holds full literal class
  strings keyed by variant (never `` `bg-${v}` `` — purge footgun). `cn.ts` = clsx + tailwind-merge.
- **Headless behavior via React Aria** (ADR-046) for Modal (Dialog/Modal/focus-trap/Esc/restore) and
  Toast (ToastRegion/queue/announcer = polite live region, errors no auto-dismiss). Styled 100% through
  tokens; watch for any library-injected color → if needed token missing, it's a finding, not a hex.
- **Progress/Step = the signature pipeline:** per-step pending/active/completed/**failed**; failed is a
  red node + visible "failed" text (never "not completed"); status-by-text-label always. Reduced-motion
  respected (active shimmer off under the global guard).
- **Minimal Vitest** (jsdom + RTL + vitest-axe) just to run `*.a11y.test.tsx` next to each component.

## Test strategy
- Per-component `*.a11y.test.tsx` asserting the §6.4 minimum (real `<button>`+`aria-busy`; label+`aria-describedby`; Badge text label; Modal role=dialog+labelled+initial focus; Table semantic + button sort headers; Toast live region + error no-auto-dismiss; Progress failed = explicit text). Run via `npx vitest run`.
- Token purity grep is the hold-#2 proof. 4.3.5b/4.3.5c is the hold-#4 anti-regression proof.

## Risks & mitigations
- **React Aria injected styles / token leak (hold #2):** style behavior, not color, through tokens; grep `--palette-`/hex in `ui/` → clean. If a primitive needs a missing token → finding (add role), never hex.
- **Purge footgun:** literal class strings only; `build` gate guards.
- **Empty State refactor breaks a gate selector (hold #4):** keep the exact headings + "Log out"; re-run 4.3.5b/4.3.5c; a drift is stop-and-escalate, not a loosen.
- **Vitest scope creep:** keep it minimal (a11y smokes only); logic tests + gate scripts stay 4.9d.
- **React 19 + RTL:** use `@testing-library/react@^16` (React 19 support); no `--force`.
