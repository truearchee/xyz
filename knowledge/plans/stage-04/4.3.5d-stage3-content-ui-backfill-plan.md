- Runtime baseline: `docker compose ps`.
- Generated client freshness: `bash scripts/generate-api-client.sh`, then `git diff --exit-code frontend/src/lib/api`.
- Contract proof: targeted source inspection plus empirical admin module creation and `module_sections` count.
