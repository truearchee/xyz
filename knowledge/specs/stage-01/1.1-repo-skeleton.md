## API changes
One endpoint only:

```http
GET /health
Response 200:
{
  "status": "ok",
  "service": "xyz-lms-backend"
}
```

No authentication. No request body. No query parameters. This endpoint must remain permanently unauthenticated because it is used by Docker Compose health checks.

## Worker / job changes
