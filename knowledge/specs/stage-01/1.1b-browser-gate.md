## Goal
Wire the root page to call `GET /health` from the browser, configure CORS so that cross-origin call succeeds, and prove it end-to-end in a real browser.

## Why now
Roadmap v2 adds a permanent browser gate: backend tests alone are not enough. Stage 1 needs browser-visible proof that the frontend route calls the backend health endpoint directly and updates from live backend state.

## Build
- Add configurable CORS origins with explicit parsing for comma-separated `CORS_ORIGINS`.
- Add CORS middleware before router registration, with origins read from `settings`.
- Rewrite the root page as a client component that polls `GET /health` through the generated API client and displays `Backend: ok` or `Backend: unreachable`.
- Keep browser calls direct to `http://localhost:8000/health`; do not proxy.
- Add backend config and CORS tests.
- Run Docker-backed backend/frontend checks and real browser verification.

## Do not build
- No new API endpoints.
- No changes to the health endpoint response or path.
- No new database tables or migrations.
- No authentication logic changes.
- No Next.js API routes or rewrites.
- No Supabase changes.
- No worker changes.
- Do not regenerate the API client.
- No design, styling, or UI component work beyond the status text.

## Verification
- `docker compose exec backend pytest tests/test_config.py -v` -> 3 passed
- `docker compose exec backend pytest tests/test_health.py -v` -> 4 passed
- `docker compose exec backend pytest` -> full suite passes
- `docker compose exec frontend npx tsc --noEmit` -> exits 0
- Browser opens `http://localhost:3000` and shows `Backend: ok`
- Browser network confirms direct `http://localhost:8000/health` and CORS header `Access-Control-Allow-Origin: http://localhost:3000`
- `docker compose stop backend` -> browser updates to `Backend: unreachable` within one poll cycle without refresh
- `docker compose start backend` -> browser recovers to `Backend: ok` without refresh
- Static checks show no direct `fetch()` in `frontend/src/app/page.tsx` and no Next.js proxy/rewrites
