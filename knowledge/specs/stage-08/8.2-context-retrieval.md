## Goal
Answers are grounded in **this one lecture's material** with full security enforcement and honest
off-lecture flagging, driven by the backend-stored `groundingStatus` (never parsed from prose).

## Build
- Server-side **context resolver** from the conversation's attached section (never trusted from the client).
- **Retrieval service** = exact pgvector scan (cosine `<=>`, NO ANN index) over the section's active
  transcript chunks, joined through the **same** published+assigned visibility gate as 4.7. First
  vector query in the codebase. Deterministic **relevance threshold**; below threshold → no context.
- Context = approved summaries + normalized chunks; **never** the raw transcript file / verbatim dumps.
- `groundingStatus` ∈ {`lecture_grounded`, `general_not_from_lecture`, `educational_redirect`,
  `context_unavailable`, `access_denied`}, backend-derived from threshold + small **structured** model
  flags (extend `AssistantAnswer`).
- UI: "not from this lecture" text label + collapsed "Where did this come from?" basis line (no raw
  transcript, no reasoning). Inline idiom.

## Do not build
- No cross-lecture / whole-module retrieval; no streaming (8.3); no modes/workspaces; no ANN index.

## Data model changes
None expected — `grounding_status` already added in migration 0032. Additive only if structured
grounding flags must be persisted.

## Security (the core — run `/cso`)
Prompt-injection (seeded "ignore instructions" treated as data), raw-transcript canary (sentinel not
in summary never surfaces — mirror `tests/test_student_summaries.py`), lost-access 404, threshold →
`general_not_from_lecture`, tampered section/module id ignored (server resolves context).

## Browser gate
Per the spec's 8.2 gate block: grounded answer, off-lecture flag, unrelated redirect, no raw
transcript, no unpublished content, unassigned → 404, other student's conversation → 404, tampered id
ignored, "Where did this come from?" safe basis line.

## Done means
Grounded to the single lecture; published/assigned/no-raw-transcript proven in the browser; off-lecture
honestly flagged; exact-scan retrieval; full suite green; browser-verified.
